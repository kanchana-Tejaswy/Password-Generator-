<h2>📌 Project Overview</h2>

<p>
The <b>Password Generator</b> is a web-based application designed to generate <b>strong, secure, and random passwords</b> to enhance user security in today’s digital environment.
</p>

<p>
It allows users to create customized passwords based on different criteria such as length, character types, and complexity, ensuring protection against common security threats like brute-force attacks and weak password usage.
</p>

<p>
This project demonstrates how simple frontend technologies can be used to build a <b>practical cybersecurity tool</b> with real-world applications.
</p>

<hr>

<h2>🔥 Key Features</h2>

<ul>
  <li>🔑 Generate strong and random passwords instantly</li>
  <li>🎛️ Adjustable password length using slider</li>
  <li>🔡 Include/exclude:
    <ul>
      <li>Uppercase letters (A–Z)</li>
      <li>Lowercase letters (a–z)</li>
      <li>Numbers (0–9)</li>
      <li>Symbols (!@#$%^&*)</li>
    </ul>
  </li>
  <li>📋 One-click copy to clipboard</li>
  <li>⚡ Real-time UI updates based on user input</li>
</ul>

<hr>

<h2>🧠 How It Works</h2>

<ul>
  <li>📌 User selects password length using a slider</li>
  <li>📌 Chooses character types using checkboxes</li>
  <li>📌 System creates a combined character pool</li>
  <li>📌 Random characters are selected using JavaScript logic</li>
  <li>📌 Final password is displayed instantly</li>
</ul>

<p>
The password is generated using <b>randomized indexing (Math.random)</b> to ensure unpredictability and security.
</p>
